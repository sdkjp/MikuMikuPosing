## 1. アプリを開く

`miku-viewer.html` ファイルをダブルクリックするか、Chrome / Edge のウィンドウにドラッグ&ドロップします。

初回は Google Fonts (DotGothic16) の読み込みに数秒かかることがあります。

**最初の画面：**

```
┌─────────────────────────────────────────────┐
│  MMP  [フォルダを開く] [ポーズリセット] ...  │  ← ツールバー
├───────┬─────────────────────────┬───────────┤
│       │                         │           │
│ ボーン│   （ここにモデルが表示） │マテリアル │
│ パネル│                         │  パネル   │
│       │                         │           │
└───────┴─────────────────────────┴───────────┘
         ↑ 最初は「MikuMikuPosing」のロゴと
           フォルダ選択ボタンが表示されます
```

## 2. モデルを読み込む

### 方法 A：ドラッグ&ドロップ（おすすめ）

Finder / エクスプローラーで「Appearance Miku」フォルダを選択し、
ブラウザ画面の中央にドロップします。

### 方法 B：ボタンから選択

1. 「📂 フォルダを選択して開く」ボタンをクリック
2. ダイアログが開いたら **「Appearance Miku」フォルダを 1 回クリック**で選択
3. **「開く / Choose」ボタン**を押す

> ⚠️ **注意**: フォルダをダブルクリックするとフォルダの中に入ってしまいます。  
> フォルダ名をクリックして**選択状態**にしてから「開く」を押してください。

モデルの読み込みには 5〜30 秒かかります。読み込みが完了すると
ミクさんが 3D で表示されます。

---

## 3. カメラを動かす

モデルが表示されたら、マウスでカメラを自由に動かせます。

| 操作 | 動作 |
|------|------|
| 左ボタンドラッグ | カメラを回転（モデルの周りをぐるっと見回す） |
| ホイールスクロール | ズームイン / ズームアウト |
| 右ボタンドラッグ | カメラを平行移動（上下左右にパン） |
| ダブルクリック | カメラをリセット（モデル全体が見える位置に戻る） |

**タブレット・スマートフォンの場合：**

| 操作 | 動作 |
|------|------|
| 1本指ドラッグ | カメラ回転 |
| 2本指ピンチ | ズーム |
| 2本指ドラッグ | 平行移動 |

---

## 4. ボーンを選択・回転させる

「ボーン」とは、モデルの骨格を構成するパーツです。ボーンを回転させることでポーズを作ります。

### 3D ビュー上でのボーン選択

1. **モデルの体をクリック** → クリックした部位のボーンが選択されます
2. 画面下部に「ポーズバー」が表示されます

```
┌──────────────────────────────────────────────────────┐
│  頭  │ 長押し＋ドラッグで回転 │ ↔↕ XY │ ↻ Z │ 完了 ✕ │
└──────────────────────────────────────────────────────┘
        ↑ 選択中のボーン名       ↑ 軸切替
```

### ドラッグで回転させる

1. モデルの回転させたい部位を **長押し**（約 0.3 秒）
2. 長押しが認識されると振動・リングアニメーションが出ます
3. そのまま**ドラッグ**すると回転します
4. 「↔↕ XY」モード: 横ドラッグ = Y 軸回転、縦ドラッグ = X 軸回転
5. 「↻ Z」モード: 横ドラッグ = Z 軸回転（ねじり）
6. 「完了 ✕」または `Esc` キーでボーン選択を解除

## 5. スライダーで細かく調整する

左サイドバーの**ボーンパネル**では、各ボーンを X・Y・Z 軸ごとに数値で正確に動かせます。

```
体 / Body Core  ▾
  センター
    X [ ──────●──── ]  0.0°
    Y [ ──────●──── ]  0.0°
    Z [ ──────●──── ]  0.0°
  上半身
    ...
```

- スライダーを左右にドラッグすると角度が変わります（−180° 〜 +180°）
- 上部の検索ボックスにボーン名を入力してすばやく絞り込めます

**よく使うボーン：**

| ボーン名 | 用途 |
|---------|------|
| センター | 体全体の高さ・左右移動 |
| 上半身 / 上半身2 | 背中を反る・前に倒す |
| 頭 | 頭を傾ける・向きを変える |
| 右腕 / 左腕 | 腕を上げ下げする |
| 右ひじ / 左ひじ | 肘を曲げる |
| 右手首 / 左手首 | 手首をひねる |

---

## 6. マテリアルの表示を切り替える

右サイドバーの**マテリアルパネル**で、パーツごとの表示/非表示を切り替えられます。

- チェックボックスを外す → そのパーツが透明になる
- 「全表示」「全非表示」ボタンで一括切替

**使用例：** 髪の毛を非表示にして、アホ毛だけ残すなど。

---

## 7. ポーズを保存・読み込む

### ポーズを保存する

1. ツールバーの「ポーズ保存」をクリック
2. 「カメラ位置も保存しますか？」ダイアログが表示されます
   - **はい** → ポーズ + カメラ角度の両方を保存
   - **いいえ** → ポーズのみ保存
3. `miku-pose.json` ファイルがダウンロードされます

### ポーズを読み込む

1. ツールバーの「ポーズ読込」をクリック
2. 保存した `miku-pose.json` ファイルを選択
3. ポーズが再現されます
4. カメラ位置も保存されていた場合は、1 秒間のアニメーションで
   そのカメラ位置にズームインします

---

## 8. VMD モーションを再生する

VMD ファイルは MikuMikuDance 用のモーションデータです。

1. ツールバーの「VMD読込」をクリック
2. `.vmd` ファイルを選択
3. 画面下部に**トランスポートバー**が表示されます

```
[ 0:00.0 / 0:30.0 ] [────────────────────────]
[ ⏮ ] [◀5s] [◀1f] [ ▶ ] [1f▶] [5s▶] [ ⏭ ] [📷] [✕]
```

| ボタン | 動作 |
|--------|------|
| ▶ / ⏸ | 再生 / 一時停止（`Space` キーでも可） |
| ⏮ / ⏭ | 最初 / 最後に移動 |
| ◀1f / 1f▶ | 1 コマ（1/30 秒）戻る / 進む（長押しで連続） |
| ◀5s / 5s▶ | 5 秒戻る / 進む（長押しで連続） |
| 📷 | 現在フレームをスクリーンショット |
| ✕ | モーションを解除してポーズをリセット |

---

## 9. スクリーンショットを撮る

ツールバーの「スクリーンショット」をクリックすると、現在の 3D 表示を
`miku-pose.png` として保存します。

**きれいな構図のコツ：**
- ボーンパネル（`B` キー）とマテリアルパネル（`M` キー）を非表示にして全画面にする
- カメラを好みの角度に調整してから撮影

## 10. 写真から全身ポーズを推定する

人物が写った写真を読み込み、体のポーズをミクさんに反映します。

1. ツールバーの「📷 全身ポーズ」をクリック
2. 全身がよく見える人物写真を選択
3. AI がポーズを解析（初回は数十秒かかります）
4. 腕・脚・頭のポーズが自動で反映されます

**よりよい結果を得るために：**
- 背景がシンプルで人物がはっきり写っている写真を使う
- 全身が写っていること（顔・手・足が切れていないもの）
- 複数人が写っている場合は最初に検出された 1 人のみ反映されます

**推定後は微調整が必要なことがあります。**  
スライダーで細かく直してください。

---

## 11. 手の写真からポーズを推定する

手だけをアップで撮った写真から、指の形とカメラ位置を再現します。

1. ツールバーの「✋ 手ポーズ」をクリック
2. 手の写真を選択
3. AI が 21 個の指ランドマークを解析
4. 指ボーン（15 本）に形が反映されます
5. カメラが手の位置に向けてアニメーション移動します

**両手が写っている場合は両方反映されます。**

**写真のコツ：**

| 良い例 | 避けるもの |
|--------|-----------|
| 明るい背景に手がはっきり写っている | 暗い・ぼけている |
| 手全体が画面内に収まっている | 手が切れている |
| 正面・斜めどちらでも OK | 手が重なりすぎて指が見えない |

## 12. よくある質問

**Q: モデルが読み込まれません**  
A: 「Appearance Miku」フォルダを丸ごと選択してください。  
個別の .pmx ファイルを選択しても動作しません。  
フォルダ選択ダイアログでフォルダ名を**1 回クリック**し「開く」を押してください。

---

**Q: テクスチャが真っ白になる**  
A: フォルダ内にテクスチャ画像（.png / .bmp）が含まれているか確認してください。  
モデルフォルダとテクスチャが別フォルダに分かれている場合は、まとめてフォルダに入れてから読み込んでください。

---

**Q: 足 IK（足首・つま先）のスライダーが動かない**  
A: 足 IK は VMD モーションを使うと正しく動作します。  
手動スライダーでは IK 解決が行われないため、見た目に反映されない場合があります。  
足・膝ボーンのスライダーを使って直接操作してください。

---

**Q: ポーズ写真機能が「人物が検出されませんでした」と出る**  
A: 以下をお試しください：
- 人物がはっきり写った明るい写真を使う
- JPEG / PNG / WebP など一般的な画像形式を使う
- インターネット接続を確認する（AI モデルの読み込みに必要）

---

**Q: 動作が重い**  
A: Chrome / Edge の使用を推奨します。  
グラフィックス負荷を下げるには「設定」→ フォグをオフにするか、
キーライトの強度を下げてみてください。

---

**Q: 保存したポーズが読み込めない**  
A: ポーズファイル（.json）が壊れているか、バージョンが異なる可能性があります。  
ファイルを開いて `"version": 1` になっているか確認してください。
