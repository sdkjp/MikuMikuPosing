# MikuMikuPosing / MMP

MMD、MQO などで用いる 3D モデルファイルをブラウザ上で **リアルタイムにポージング** できる Web アプリケーションです。  
スマートフォン、タブレット、PC すべてのデバイスに対応しており、インストール不要で利用できます。

---

## ✨ 主な機能

### 🎬 モデル読み込み
- **PMX ファイル対応** — Vocaloid モデルに最適化
- **ドラッグ＆ドロップ** または **フォルダ選択** で簡単読み込み
- テクスチャ、Toon シェーダーの自動認識
- BMP / SPA / SPH 形式画像を自動で PNG に変換（ブラウザ互換性向上）

### 🦴 ボーン操作（インタラクティブポージング）
- **直感的なドラッグ操作** — モデル上をクリック＆ドラッグで即座に回転
- **ボーンパネル** — 階層的なボーン一覧から細かく選択
- **3D 回転ギズモ** — X / Y / Z 軸の個別制御
- **スライダー操作** — 各軸の角度を数値で微調整
- **即座プレビュー** — 変更が即座に 3D ビューに反映

### 🎨 マテリアル表示制御
- モデルのマテリアル一覧を表示
- チェックボックスで表示 / 非表示を切り替え
- マテリアルカラープレビュー

### 🎞️ モーション再生（VMD サポート）
- **VMD ファイル読み込み** — MMD 標準モーションフォーマット対応
- 再生 / 一時停止 / シーク操作
- フレーム単位での前後移動
- 時刻表示と進捗スクラバー

### ⚙️ 環境設定
- **スカイカラー** — 背景色のカスタマイズ
- **グラウンドカラー** — 床色の変更
- **照明調整** — アンビエント光、キーライト、シャドウ強度の制御
- **フォグ効果** — 奥行き感の有効 / 無効

### 💾 データ保存
- **ポーズ保存 / 読み込み** — JSON 形式でポーズを永続化
- **スクリーンショット** — 現在の表示を PNG で保存
- **フレーム撮影**（モーション再生中）

### 📱 デバイス対応
- **デスクトップ** — マウス / タッチパッド / トラックボール対応
- **タブレット / スマートフォン** — タッチ操作最適化

## 🚀 使用方法

### 基本的な流れ

1. **モデルフォルダを開く**
   - ツールバーの「📂 フォルダを選択して開く」ボタン、またはドラッグ＆ドロップ
   - フォルダ内に `.pmx` ファイルが含まれていることを確認

2. **モデルが読み込まれたら**
   - 左側パネル：ボーン一覧
   - 右側パネル：マテリアル一覧
   - 中央：3D ビューポート

3. **ポージング**
   - 3D モデルを **左ドラッグ** で視点回転
   - ボーンを **クリック** して選択
   - **長押し＋ドラッグ** で回転操作
   - スライダーで細かく角度調整

4. **モーション再生**
   - VMD ファイルを読み込み
   - 再生ボタンで再生開始
   - 時刻スクラバーでシーク

### キーボード操作

| キー | 動作 |
|------|------|
| **Space** | VMD 再生 / 一時停止 |
| **マウスホイール** | ズーム |
| **右ドラッグ** | 視点移動（パン） |

### マウス / タッチ操作

| 操作 | 動作 |
|------|------|
| **左ドラッグ** | 3D 視点回転（OrbitControls） |
| **ホイールスクロール** | ズームイン / アウト |
| **右クリック + ドラッグ** | 視点パン |
| **長押し（260ms）+ ドラッグ** | ボーン回転 |
| **タップ** | ボーン選択 / パネル操作 |

## 🛠️ 技術仕様

### 使用技術
- **Three.js 0.160.0** — 3D グラフィックスライブラリ
- **MMDLoader** — Three.js 拡張（PMX / PMD パーサー）
- **OrbitControls** — カメラ制御
- **Web APIs** — File API、Canvas、WebGL

### ファイル対応
| フォーマット | 対応状況 | 説明 |
|------------|--------|------|
| **PMX** | ✅ 完全対応 | Vocaloid モデル推奨 |
| **PMD** | ✅ 完全対応 | 旧式 MMD モデル |
| **VMD** | ✅ 完全対応 | モーションファイル |
| **BMP** | ✅ 自動変換 | PNG に変換して読み込み |
| **PNG / JPG** | ✅ 対応 | テクスチャ形式 |
| **Toon** | ✅ 対応 | Toon シェーディング |

### パフォーマンス
- **レスポンシブ設計** — 解像度に応じて自動調整
- **PixelRatio 最適化** — `Math.min(devicePixelRatio, 2)` で Retina 対応
- **Shadow Mapping** — PCFSoftShadowMap による柔らかい影
- **ACES Filmic Tone Mapping** — 映画的な色空間

### パネル機能

#### 左パネル（ボーン操作）
- **グループ展開** — アコーディオン形式で階層管理
- **グループ一覧**
  - 体 / Body Core
  - 頭 / Head（目、口含む）
  - 左右腕 / Arms + 手指
  - 左右足 / Legs + 足 IK
  - ネクタイ、髪（前髪、ツインテール）
  - スカート、袖

#### 右パネル（マテリアル表示）
- マテリアルカラースウォッチ
- チェックボックスで表示 / 非表示を切り替え

#### 下部パネル（ポーズバー）
- 選択中のボーン名
- X / Y / Z 軸選択ボタン
- 「完了」ボタンで選択解除

#### 下部パネル（VMD トランスポート）
- 再生 / 一時停止 / シーク
- フレーム単位での移動（±1f, ±5s）
- 時刻表示 / スクラバー
- スクリーンショット（フレーム撮影）

## 📝 ショートカット / Tips

### デスクトップ

- **Z ボタン** — ロール（Z 軸回転）に変更

## 🐛 トラブルシューティング

### モデルが読み込まれない
- `.pmx` ファイルが含まれているか確認
- フォルダ全体を選択（ファイルではなく）
- ブラウザコンソール（F12）でエラーを確認

### テクスチャが表示されない
- テクスチャファイルがフォルダに含まれているか確認
- BMP 形式の場合、自動的に PNG に変換されます
- 大文字・小文字の一致を確認

### パフォーマンスが低い
- 高ポリゴンモデルの場合、ビューを制限
- ウィンドウを小さくするか、解像度を下げる
- 他のブラウザタブを閉じる

### タッチ操作がうまくいかない
- 長押し時間は 260ms に設定
- ビジュアルリングが表示されたらドラッグを開始
- スタイラスペン使用時は精度が上がります
