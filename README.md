# MikuMikuPosing (MMP)

ブラウザだけで動く MikuMikuDance PMX モデル ポージングツールです。
サーバー不要・インストール不要。HTML ファイルをブラウザで開くだけで使えます。

---

## 特徴

| 機能 | 説明 |
|------|------|
| PMX モデル読込 | フォルダをドラッグ&ドロップ、またはダイアログで選択 |
| **自動読み込み** | **2回目以降は起動時にモデル・VMD を自動再読み込み（Chrome）** |
| カメラ操作 | ドラッグで回転・ホイールでズーム・右ドラッグで平行移動 |
| ボーン操作 | クリックでボーン選択 → 長押し＋ドラッグで回転、またはスライダー直接操作 |
| マテリアル表示切替 | 右サイドバーのチェックボックスで個別に表示/非表示 |
| ポーズ保存・読込 | JSON ファイルとして書き出し / 読み込み（カメラ位置も保存可） |
| VMD モーション | VMD ファイルを読み込んで再生・スクラブ・コマ送り |
| スクリーンショット | 現在の画面を PNG で保存 |
| 全身ポーズ推定 | 人物写真から体のポーズを自動推定（MediaPipe PoseLandmarker） |
| 手ポーズ推定 | 手の写真から指の形を自動推定＋カメラ位置を合わせる（MediaPipe HandLandmarker） |
| **手・耳ビューア** | **`hand-viewer.html` — 漫画家・イラストレーター向け、iPhone/iPad 最適化** |

---

## 動作環境

- **Chrome 90 以上** または **Edge 90 以上**（推奨）
- Firefox / Safari でも動作しますが、一部機能に制限がある場合があります
- インターネット接続が必要（Three.js / MediaPipe を CDN から読み込むため）
- ポーズ写真機能の初回使用時は数秒〜数十秒のダウンロードが発生します

## 使い方（クイックスタート）

```
1. miku-viewer.html をブラウザで開く
2. 「フォルダを選択して開く」をクリック → モデルフォルダを選択
3. モデルが表示されたらポージング開始
```

> **フォルダ選択のコツ**  
> ダイアログが開いたら「Appearance Miku」フォルダを 1 回クリックして選択し、
> 「開く / Choose」ボタンを押してください（ダブルクリックでフォルダ内に入らないように）。

---

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Space` | VMD 再生 / 一時停止 |
| `R` | ポーズをリセット |
| `F` | カメラをリセット（モデル全体にフィット） |
| `B` | ボーンパネルを開閉 |
| `M` | マテリアルパネルを開閉 |
| `Esc` | ボーン選択を解除 |

## ポーズファイル仕様 (JSON)

```json
{
  "version": 1,
  "model": "Appearance Miku",
  "bones": [
    { "index": 0, "name": "センター", "rx": 0.0, "ry": 0.0, "rz": 0.0 }
  ],
  "camera": {
    "position": [0, 14, 32],
    "target": [0, 12, 0]
  }
}
```

`camera` フィールドはポーズ保存時に「カメラ位置も保存しますか？」に「はい」と答えた場合のみ含まれます。

```
Project/0515/
├── miku-viewer.html        ← メインアプリ（全身ポージング）
├── hand-viewer.html        ← 手・耳ビューア（漫画家/イラストレーター向け）
├── README.md               ← このファイル
├── TUTORIAL.md             ← 初心者向けチュートリアル
├── Appearance Miku/        ← モデルデータ（別途用意）
│   ├── Appearance Miku.pmx
│   ├── Amiku1.png
│   └── ...
└── （VMD モーションフォルダ）/
    └── *.vmd
```

> **自動読み込みについて**  
> Chrome で `showDirectoryPicker()` または `showOpenFilePicker()` を使ってフォルダ・VMD を開くと、  
> IndexedDB にハンドルが保存されます。次回起動時は許可が残っていれば **自動でモデル＆VMD を読み込みます**。  
> 許可が切れた場合は「↩ 前回のフォルダを再読み込み」ボタンが表示されます。

## 手・耳ビューア (hand-viewer.html)

漫画家・イラストレーターが手や耳のポーズ参考に使うための専用ページです。

### 機能

| 機能 | 説明 |
|------|------|
| 手ポーズ | 指 5 本 × 3 関節をタブで切り替えて操作 |
| 耳ビュー | モデル頭部を耳のアップで表示、解剖学ラベル付き |
| 右/左切替 | ワンタップで右手⇄左手、右耳⇄左耳 |
| プリセット | 開く・握る・指差し・ピース・OK・ピンチなど 8 種 |
| 長押し+ドラッグ | 3D ビュー上で直接ボーンを回転（タッチ対応） |
| スクリーンショット | 現在の画面を `hand-pose.png` で保存 |
| 自動読み込み | メインビューアと同様、前回フォルダを自動復元 |

### 操作方法 (iPhone/iPad)

1. `hand-viewer.html` をブラウザで開く
2. モデルフォルダを選択（初回のみ）
3. 画面下部で**指タブ**を選択
4. スライダーで各関節を調整、または**長押し+ドラッグ**で直感操作
5. プリセットボタンで一発ポーズ変更

## 制限事項・注意点

- **物理演算なし**: 髪・スカートの揺れはシミュレーションされません
- **IK ボーン**: 足 IK は VMD 再生時のみ動作します（スライダーでの直接操作は反映されません）
- **ポーズ写真推定**: 近似値のため、細部は手動スライダーで調整してください
- **カメラマッチング**: レンズ焦点距離が不明なため完全一致は困難です
- **対応モデル**: PMX 2.0 形式。Appearance Miku 以外のモデルでも読み込めますが、ボーン番号が異なる場合は写真ポーズ推定の対応ボーンがズレることがあります
